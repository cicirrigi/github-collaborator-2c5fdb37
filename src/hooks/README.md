# Hooks Directory

This directory contains custom React hooks for the Vantage Lane 2.0 project.

## Structure

- Custom hooks should follow the `use` prefix naming convention
- Each hook should be properly typed with TypeScript
- Include JSDoc documentation for complex hooks

## Examples

- `useLocalStorage.ts` - Local storage hook
- `useDebounce.ts` - Debounce hook
- `useApi.ts` - API data fetching hook
