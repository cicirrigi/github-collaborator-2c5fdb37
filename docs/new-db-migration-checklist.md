## **ЁЯЪи CRITICAL DIFFERENCES - NEW vs OLD DB**

| Feature                    | OLD DB            | NEW DB                         |
| -------------------------- | ----------------- | ------------------------------ |
| **Initial payment status** | `pending`         | `unpaid`                       |
| **Quote expiration**       | `expires_at`      | `quote_valid_until`            |
| **Booking reference**      | Manual generation | `generate_booking_reference()` |
| **Vehicle IDs**            | UUID              | TEXT                           |
| **trip_configuration_raw** | Optional          | NOT NULL (required)            |
| **Payment flow**           | CreateтЖТpending    | CreateтЖТunpaidтЖТpending          |

### **Files Modified:**

- тЬЕ `/src/app/api/bookings/route.ts` - Updated for NEW DB schema
